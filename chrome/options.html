Scripture Tips Settings


    ST_REFERENCE : Full scripture reference (eg: James 3:13-15).
    ST_BOOKNAME : The name of the book (eg: James).
    ST_CHAPTER : The chapter (eg: 3).
    ST_VERSE : The verses if applicable (eg: 13-15).
    ST_TRANSLATION : The translation as defined above.
    Examples:
    https://www.biblegateway.com/passage/?search=ST_REFERENCE&version=ST_TRANSLATION
    OR
    https://biblehub.com/ST_TRANSLATION/ST_BOOKNAME/ST_CHAPTER:ST_VERSE
